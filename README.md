## Como rodar

1. Instale as deps

```bash
  yarn
```

2. Dev server com hot reload na http://localhost:3000.

```bash
yarn dev
```

## Comandos de testes

- Roda todos os testes
  ```bash
  yarn test
  ```
- Roda unit tests com watch
  ```bash
  yarn test:unit
  ```
- Roda unit tests com coverage
  ```bash
  yarn test:unit:coverage
  ```

## Comando de build

```bash
yarn build
```
